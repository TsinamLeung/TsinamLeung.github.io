## TypeDuck

TypeDuck is a Cantonese input method project with multilingual dictionary support, developed to assist Cantonese learners from diverse ethnic backgrounds and planned for introduction in Hong Kong primary and middle schools. My work focused on the engineering and lexical-resource side of the project: IME algorithm server work, dictionary interface development, and refinement of Cantonese lexical entries for practical input use.

### Contributions

- Built and deployed the IME algorithm server.
- Designed and implemented the dictionary user interface.
- Refined Cantonese lexical entries and removed obsolete or rare terminology where appropriate.
- Worked with dictionary and N-gram processing for input-method use.

### Relevance

This project connects language-resource work with usable language technology: turning lexical data into an input method and dictionary interface that can support Cantonese learners and users.
